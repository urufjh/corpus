 Корпус жестов и речи

Веб-приложение для работы с мультимодальным корпусом данных, содержащим видео и аннотации речи и жестов. Позволяет искать по словам, леммам и жестам, просматривать аннотации на временной шкале, а также получать статистику по жестам в виде барплота и таблицы.

 Возможности

- Поиск по словам - находит все вхождения слова в тексте аннотаций
- Поиск по леммам - ищет по нормальной форме слова (до этого добавляет леммы всех слов из слоя Words разметки)
- Поиск по жестам - фильтрация по типу жеста (Depictive, Pragmatic, Pointing и др.)
- Временная шкала - интерактивное отображение аннотаций с возможностью перемотки видео
- Статистика жестов - график распределения жестов по типам для каждого видео
